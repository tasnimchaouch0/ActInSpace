# GreenSignal (AI-Driven Vegetation Monitoring)

## Project Structure
- `frontend/` – React app (map, dashboard, UI)
- `backend/` – Node.js/Express API
- `ai_service/` – Python ML/data processing
- `db/` – Database setup (PostgreSQL + PostGIS)

## Quick Start
- See each folder for setup instructions.
- Docker and CI/CD included for easy deployment.
